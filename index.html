■ 개요 및 설명

202X년대 유행한 특정 M게임의 세계 생성 코드인 SEED(시드)를 고르라는 숏츠 영상에서 타고 나간 재난으로 확인 됨. 영상에 사용된 M게임 말고도 여러 게임을 통해 이동하며 특정 게임을 기반으로 한 가상 세계를 만드는 것이 특징.

1. 생성 조건

  • 짧은 시간 (1s~15s) 동안 일정 이상의 트래픽이 모일 경우 20m 반경에 있는 PC, 전자기기, 게임기 등에서 발생한다.
  • PC에 다운로드 되어있는 게임 and 트래픽에 가장 가까운 물체, 생명체, ■■■ 등과 일부 혼합되어 세계가 만들어진다. 이때 게임의 바탕으로 혼합된 물체, 생명체, ■■■을 SEED 라고 지칭한다.
  • 세계는 기본적으로 바탕이 된 게임과 SEED로 인해 존재하지만, 세계 제작 과정 중 부족한 데이터는 외부(현실) 에서 끌어온다.
  • 세계 제작 시 외부에서 끌어온 데이터 중, 형체가 소멸된 물건, 사람 등은 구축된 가상 세계 안에 내부 리소스 파일로 재생성되묘, 게임 내에서 채운 데이터의 역할에 따라 기능한다. (변환 된 데이터 종류로는 맵, 파괴 오브젝트, 인벤토리 아이템, NPC 캐릭터, 에너미 캐릭터 등이 있다.)

2. 생성 구조

세계가 생성될 때, 권장 목표 수치보다 내부 데이터가 부족할 경우 전기 신호, 통신망 등을 통해 현실에 나타난다. 사이트의 팝업, 메세지, 알림, 문자, 전화 등 여러가지 형태로 나타나나 그 형태는 SEED가 해당 존재를 흡수하기 수월하도록 실제와 비슷한 형태인 것으로 확인되었다.

⚠️ 주의 사항: 외부통신망을 통해서 접속한 사람들의 경우 몸은 현실에 남으나 정신만 가상 세계로 이월 되는 것으로 보인다. 통신망을 통한 접속에서 강제로 대상자를 깨우려고 할 경우 제정신이 아니거나, 대화나 의사소통이 불가능한 상태에 들어갔다는 기록이 있으므로 각별한 주의가 필요함.

3. 진입 제한

세계 유지에 필요한 최소 데이터가 채워진 이후 추가 진입은 불가능. 그렇기 때문에 일정 데이터를 비우면 동일 괴담에 재진입 가능할 것이라는 가설이 있다.

[수정 사항]

0110PSYA.1974.파01 재난에서의 출동 기록에서 정리된 진입 방법을 추가했습니다.

[전제 조건]
  • 괴담의 중심이 되는 장비에 접근 가능한 상태여야 함
  • 관리자 권한으로 게임이 실행이 가능하거나, 편집이 가능한 상태에 있어야 함
  • SEED와 세계가 충분히 안정성이 있는 상태여야 함

해당 전제 조건이 전부 갖춰진 상태에서, 아래의 방법을 통해 진입한다.

1. 서버가 존재할 경우: 디버그 모드로 전환해 서버에 전송하는 데이터에 재진입 할 대상과 연관된 일부 파일을 집어넣는다. 이후 데이터를 서버에 전송한 이후, 처음 진입했던 매개체를 통해 한 번 더 해당 게임과 접촉한다. (접촉 방식: 메세지, 팝업, 알림, 문자, 전화 등)

2. 싱글 플레이 월드일 경우: 월드 데이터의 일부분을 삭제하거나 변경해 일부분 오류를 일으킨다. (※ 주의: 데이터를 변경할 때 건물 또는 오브젝트 파일에 넣지 않도록 할 것) 이후 데이터 무결성 검사를 실행한다. 무결성 검사 중 파일 업로드 실패 알림이 온다면 개발사가 서버에 저장한 초기 데이터를 다시 다운로드한다. 해당 재난을 통해 만들어진 세계라면, SEED를 중심으로 세계 생성시의 내부 데이터 수급 현상을 다시 일으킨다.

※ 위의 방법은 비슷한 재난 0110PSYA.1974.파01 에서 실행 가능하다는 것이 확인 된 내용임.
